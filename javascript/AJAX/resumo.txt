AJAX significa Asynchronous JavaScript and XML, ou JavaScript e XML Assíncronos, em bom português. Ele é um conjunto de técnicas de desenvolvimento voltado para a web que permite que aplicações trabalhem de modo Assíncrono, processando qualquer requisição ao servidor em segundo plano.

O JavaScript é uma linguagem de programação bem conhecida que gerencia conteúdos dinâmicos de um site e permite a interação dinâmica com o usuário. O XML(eXtensible Markup Language), como o nome sugere, é uma variação de linguagem de marcação no estilo do HTML. Enquanto o HTML é utilizado para exibir dados, o XML os armazena e transmite.

Qualquer aplicação que use AJAX pode enviar e receber dados do servidor sem precisar recarregar a página inteira. Isso torna o processo de multitarefas mais simples. Sempre que você encontrar uma situação em que duas operações funcionam simultaneamente, uma em execução e a outra ociosa, é bem provável que o AJAX esteja envolvido.

Um exemplo prático do uso de AJAX é a ferramenta de sugestões de pesquisa do Google. Ela ajuda a completar as palavras que você digita em tempo real enquanto a página permanece estática.
